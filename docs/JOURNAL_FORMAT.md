# Daily files (optional, local)

The agent can write dated files under `journal/` (ignored by git in this repo so you don’t leak personal content).

Suggested names:

- `journal/2026-04-03-plan.md` — morning commitment (bullets, optional times).
- `journal/2026-04-03-evening.md` — evening self-report + short AI summary.

Use **one timezone** consistently for the date in the filename (e.g. where you live).
